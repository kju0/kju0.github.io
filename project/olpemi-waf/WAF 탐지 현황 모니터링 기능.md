## 🎯 목표
**WAF 탐지 현황 모니터링 기능 구현**  
- WAF 탐지 로그 → MySQL → 모니터링 웹
- WAF 탐지 로그 → MySQL 중간 과정
	- 로그 Json 저장
	- 로그 변경사항 추적
	- 비동기 처리(RabbitMQ, Logstash)

    - WAF에서 YAZL 라이브러리를 사용할 수 있도록 링킹 작업
